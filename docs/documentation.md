# Dokumentasi Proyek

Halaman ini menjadi pintu masuk semua dokumen teknis.

## Daftar Dokumen
- `docs/README.md`  ringkasan dokumentasi
- `docs/quickstart.md`  setup lokal cepat
- `docs/architecture.md`  arsitektur komponen
- `docs/api_reference.md`  endpoint API terbaru
- `docs/operations.md`  runbook operasional
- `docs/deployment.md`  deployment environment server
- `docs/ssh.md`  akses SSH aman

## Prinsip Update Dokumen
- Update dokumen setiap ada perubahan endpoint, flow bisnis, atau command utama.
- Hindari menaruh kredensial/API key di markdown.
- Jadikan Swagger (`/docs`) sebagai sumber schema API paling aktual.

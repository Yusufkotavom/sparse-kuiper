# UI Change: Simplify & Unify

## Workflow
1. Identifikasi komponen/halaman yang duplikatif.
2. Satukan pattern spacing, typography, warna, dan interaksi.
3. Pertahankan alur navigasi utama agar pengguna lama tidak bingung.
4. Validasi visual melalui screenshot sebelum/sesudah.

## Checklist Minimum
- [ ] Tidak menghapus fungsi lama tanpa pengganti jelas.
- [ ] Komponen UI yang sama memakai style system yang sama.
- [ ] Empty/loading/error states konsisten.
- [ ] Ada catatan perubahan UI yang berdampak pada user.

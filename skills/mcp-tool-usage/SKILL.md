# MCP Tool Usage

## Workflow
1. Mulai dari resource MCP yang tersedia sebelum membuat asumsi.
2. Gunakan resource templates hanya jika parameter dibutuhkan.
3. Simpan artifact penting (log, screenshot, output test).
4. Hindari operasi besar yang tidak perlu (context hygiene).

## Checklist Minimum
- [ ] Resource discovery dilakukan di awal.
- [ ] Output kritikal disimpan sebagai artifact.
- [ ] Tool dipakai sesuai kebutuhan task.
- [ ] Ada fallback jika resource tidak tersedia.
